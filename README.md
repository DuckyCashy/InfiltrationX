# InfiltrationX

InfiltrationX is a Roblox script that changes our visual gameplay.
---

## ✨ Features

### Visuals
- **Stretched Resolution** (Horizontal + Vertical)
- **Full Bright**

### ESP
- Toggle ESP
- Team Check
- Multiple ESP Types:
  - Hitbox ESP
  - Hitbox ESP V2
  - Box ESP
  - Tracers

### Environment
- **Custom Skybox** (with presets + custom ID)
- **Atmosphere Control**
  - Enable/Disable
  - Thickness Slider (0-10)
  - Color Presets

### Performance
- Disable Shadows
- Disable Sunrays
- Disable Post Effects
- Remove Fog
- Disable Textures & Particles
- Low Texture Quality
- Low Terrain Quality
- Low Graphics Mode
- FPS Cap (Slider + Textbox)
- **Disable/Enable 3D Rendering**

### Info & Utilities
- Executor Info
- Player Count + Friend Count
- Rejoin + Server Hop

### UI Settings
- Theme Manager
- Save Manager
- Custom Keybinds
- DPI Scale
- Corner Radius
- Reset Options

---

## 📥 How to Use

1. Execute the script using your preferred executor (Synapse X, Krnl, Fluxus, etc.)
2. Press **G** to open the menu
3. Navigate through the tabs to customize your experience

---

## ⚙️ Configuration

- All settings are saved automatically via SaveManager
- You can create multiple configs in the UI Settings tab
